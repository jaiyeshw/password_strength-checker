# Password Strength Checker

A beginner-friendly Python cybersecurity project that checks password strength based on length, character variety, common patterns, and basic security rules.

## Features

- Checks password length
- Detects uppercase and lowercase letters
- Detects numbers and special characters
- Warns about common passwords
- Warns about predictable sequences like 123, abc, and qwerty
- Hides password input using getpass
- Allows checking multiple passwords

## How To Run

```bash
python password_strength_checker.py
